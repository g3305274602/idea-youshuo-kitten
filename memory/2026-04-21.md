# 2026-04-21 介面／模組變更紀錄

## SpacetimeDB 模組（`spacetimedb/src/index.ts`）

- **新增表 `capsule_favorite`（`capsule_favorite`）**：收藏「膠囊本體」快照；欄位 `capsuleId`、`collectorIdentity` 使用 `.index('btree')`（符合 TS SDK 2.x，避免舊式 `indexes` 缺少 `accessor` 編譯失敗）。
- **新增 reducers**：`favorite_capsule`、`unfavorite_capsule`（驗證登入、膠囊存在、非作者、未刪、已到開啟時間、防重複收藏／刪除收藏）。
- **`send_capsule_message`**：維持既有參數相容，廣場相關行為收斂（送出膠囊不再自動貼廣場；與撰寫流程分離）。
- **指令**：已執行 `npm run spacetime:generate` 產出 `capsuleFavorite`、`favoriteCapsule`、`unfavoriteCapsule` 等 client bindings。

## 前端（`src/App.tsx`）

- **註冊**：表單移除年齡欄位；呼叫 `registerAccount` 時傳 `ageYears: undefined`（codegen 型別要求鍵存在）。
- **登入後年齡**：`needsAgeGate` 涵蓋缺漏／無效年齡；維持滾動清單選年齡後 `setAgeYears`。
- **定向 vs 秘密膠囊撰寫**：膠囊模式移除「個人空間公開、waitlist 勾選、順手貼牆」區塊；送出膠囊時廣場相關參數固定為關；定向仍保留排程、waitlist、貼牆選項。
- **收件匣**：未到開啟時間時，僅在 `isWaitListVisible` 為真時顯示於收件列表（其餘對收件人隱藏）。
- **抽膠囊池**：排除已廣場收藏、已膠囊本體收藏、以及自己已開「訪客線」的膠囊（`capsule_private_for_me` 上 `threadGuestIdentity === 自己`）。
- **收藏**：接上 `favoriteCapsule`／`unfavoriteCapsule`；膠囊彈窗在無 `square_post` 時仍可「藏進心底」；**心底**左欄與詳情合併廣場收藏與膠囊收藏（統一 key：`sq:`／`cap:`）。
- **React Hooks**：`unifiedFavoriteItems`（`useMemo`）與收藏 id 校正（`useEffect`）自 login/register 的 early return 之後遷到**之前**，避免「登入頁少跑 hooks／儀表板多跑 hooks」導致順序錯誤與 `Rendered more hooks than during the previous render`。
- **建置**：`npm run build` 通過。
